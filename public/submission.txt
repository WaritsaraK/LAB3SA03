Game จับคู่ภาพ

commit part 1
    ทำการสร้างตารางขนาด 4x4 สำหรับใส่รูปภาพที่ต้องการจับคู่
    โดยทำการสร้างไฟล์ Cards.js และCard.js และแก้ไข้โค้ดภายในไฟล์ App.js และ index.js

commit part 2
    ทำการใส่รูปภาพที่ต้องการจับคู่ ลงตามช่องตารางที่สร้างไว้ และทำการสุ่มสลับตำแหน่งของรูปภาพ
    โดยสร้าง Folder สำหรับเก็บรูปภาพไว้ภายใน Public จากนั้นนำเข้ารูปที่ต้อง
    การใส่รูปและสลับตำแหล่งทำผ่าน function Cards
    ในส่วน const [items, setItems] = useState([]) 
    การใส่ตำแหน่งที่ของรูป  เช่น { id: 1, img: '/pics/Bird.png', stat: "" } รูปภาพ Bird อยู่ที่ folder pics
    การสลับตำแหน่งของรูป โดยคำสั้ง .sort(()=> Math.random()-0.5)

commit part 3
    การทำให้แผ่นป้ายของตารางหมุนปิดเปิด โดยจะทำการแก้ไขปรับปรุงภายใน index.css
    กำหนดให้ช่องตารางหมุนตามแนวแกน y 180 องศา และย่อขนาดของรูปภาพลง

commit part 4
    ปรับให้สามารถคลิกเลือกรูปภาพได้
    เพิ่ม function handleClick ภายใน Cards.js ในส่วน function Cards
    
commit part 5
    การตรวจสอบการจับคู่รูปภาพ
    เพิ่ม function check ภายใน Cards.js ในส่วน function Cards
    โดยจะเปรียบเทียบรูปที่คลิกครั้งแรกกับครั้งที่สอง ถ้าหากเหมือนกันจะปรากฎสี และคงอยู่ไม่พลิกกลับ ถ้าไม่ถูกต้องจะแสดงสีแล้วทำการพลิกกลับเพื่อเลือกใหม่
    ใน index.css กำหนดสีให้กับการจับคู่
    ถ้าหากถูกต้องเป็น สีฟ้า
    ถ้าหากผิดจะเป็น สีแดง
